== Opening Applications ==

Win+Enter   :: Open terminal
Win+F1      :: Open terminal
Win+F2      :: Open firefox
Win+F3      :: Open nautilus
Win+F4      :: Open thunderbird


== Moving Around ==

Win+d       :: Open menu (for opening applications)
Win+Arr     :: Change focus to another window
Win+a       :: Change focus to parent container
Win+Sft+Arr :: Move window
Win+v       :: Split vertically (note blue border)
Win+h       :: Split horizontally (note blue border)
Win+Sft+q   :: Close window
Win+Delete  :: Close window


== Changing the Container Layout ==

Win+e       :: Split horizontally/vertically
Win+s       :: Stack windows
Win+w       :: Tabbed windows
Win+f       :: Toggle fullscreen window
Win+Sft+Spc :: Toggle floating window

Resize boundaries with mouse


== Using Workspaces ==

Win+Num     :: Go to workspace NUM
Win+Sft+Num :: Move window to workspace NUM


=== Session Management ==

Win+Sft+r   :: Restart i3 session in place
Win+Esc     :: Power Management Mode
  +l        :: lock
  +e        :: logout
  +r        :: reboot
  +h        :: shutdown
